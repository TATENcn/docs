**Nuxt Studio** 模块是一个基于浏览器的界面，用于直接在生产环境中编辑您的 Nuxt Content 网站。在您部署的网站上通过 GitHub、GitLab 或 Google 身份验证访问它，并开始管理内容，无需任何本地开发工具。

::tip{to="https://nuxt.studio/introduction"}
浏览 Nuxt Studio 文档以了解如何安装该模块。
::

:video{controls loop src="https://res.cloudinary.com/nuxt/video/upload/v1767647099/studio/studio-demo_eiofld.mp4"}

**Studio 编辑器**允许您在生产网站上完全从浏览器管理内容。无需本地开发工具、Git 命令或终端访问。对于希望在熟悉的环境中编辑和预览更改的内容团队来说，这是理想的选择。

## 在生产环境中为您的 Nuxt Content 网站提供可视化编辑

Nuxt Studio 为基于 Nuxt Content 构建的网站提供**直接在生产环境中的可视化编辑功能**。

最初作为独立的付费平台推出，如今 Studio 已成为一个**免费、开源且可自托管的 Nuxt 模块**。它使您的整个团队（无论是开发者还是非技术人员）都能安全地创建和更新内容，而无需离开您的线上网站。

## 当前功能

### ✨ **TipTap 可视化编辑器**

支持完整 MDC 组件的富 Markdown 编辑器。

### 💻 **Monaco 代码编辑器**

高级代码编辑器，用于编辑 Markdown（MDC）、YAML 和 JSON 文件（如果您希望编辑原始代码）。

### 📝 **表单式编辑器**

根据集合模式自动生成表单，用于编辑 YAML、JSON 和页面前置元数据（frontmatter）。

### 🎨  **Vue 组件属性编辑器**

可视化界面，可直接从编辑器中编辑 Vue 组件的属性。

### 🔄 **实时预览**

即时在您的生产网站上预览内容更改。

### 🔐 **多提供商身份验证**

通过 GitHub、GitLab 和 Google 提供的安全 OAuth 身份验证。

### 🔑 **自定义身份验证**

提供工具以实现自定义身份验证流程（密码、SSO、LDAP 等）。

### 📝 **文件管理**

在 [content/]{.s2} 目录中创建、编辑、重命名和删除内容文件。

### 🖼 **媒体管理**

集中式媒体库，支持 JPEG、PNG、GIF、WebP、AVIF、SVG 等格式。

### 🌳 **Git 集成**

直接从生产环境提交内容更改，并依靠您的 CI/CD 流水线进行部署。

### 🚀 **开发模式**

使用 Studio 界面直接从本地文件系统编辑内容和媒体文件。

### 🌍 **国际化**

全面的 i18n 支持，涵盖 17 种语言：阿拉伯语（AR）、保加利亚语（BG）、德语（DE）、英语（EN）、西班牙语（ES）、波斯语（FA）、芬兰语（FI）、法语（FR）、印尼语（ID）、意大利语（IT）、日语（JA）、荷兰语（NL）、波兰语（PL）、巴西葡萄牙语（PT-BR）、乌克兰语（UA）、简体中文（ZH）、繁体中文（ZH-TW）。
