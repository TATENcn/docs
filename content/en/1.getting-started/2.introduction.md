欢迎来到 **Docus**，一个基于 [Nuxt UI](https://ui.nuxt.com) 构建的完全集成的文档解决方案。

## 什么是 Docus？

Docus 是一个基于 [UI 文档模板](https://docs-template.nuxt.dev/)的主题。虽然视觉样式已经开箱即用，但您应该专注于使用 [Nuxt Content](https://content.nuxt.com) 提供的 Markdown 和 [MDC 语法](https://content.nuxt.com/docs/files/markdown#mdc-syntax)来编写内容。

## 核心特性

此主题包含一系列旨在改进文档管理的功能：

- **基于** **[Nuxt 4](https://nuxt.com)** **构建**：利用最新的 Nuxt 框架以获得最佳性能。
- **使用** **[Nuxt UI](https://ui.nuxt.com)** **构建**：集成了一套全面的 UI 组件。
- **通过** **[Nuxt Content](https://content.nuxt.com)** **支持** **[MDC 语法](https://content.nuxt.com/usage/markdown)**：支持 Markdown 与组件集成以实现动态内容。
- **兼容** **[Nuxt Studio](https://content.nuxt.com/docs/studio)**：可视化地编写和编辑您的内容。无需 Markdown 知识！
- **自动生成侧边栏导航**：根据内容结构自动生成导航。
- **全文搜索**：包含用于内容发现的内置搜索功能。
- **优化的排版**：提供精致的排版以增强可读性。
- **深色模式**：根据用户偏好提供深色模式支持。
- **丰富的功能**：探索主题以充分了解其功能。
